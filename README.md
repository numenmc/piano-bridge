# piano-bridge-venv
Play Minecraft with the piano.

## Requirements
```
pip install pyautogui
pip install mido
pip install python-rtmidi
pip install pyfluidsynth
```

Must download fluidsynth: https://github.com/FluidSynth/fluidsynth/releases
Place fluidsynth /bin into C:/tools/fluidsynth/bin

Download "FluidR3_GM.sf2" from https://member.keymusician.com/Member/FluidR3_GM/ and place in assets/

If you get a MIDI error "the port is already in use" you may need to use https://www.tobias-erichsen.de/software/loopmidi.html and Midi-OX
